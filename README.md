# Java Learning Projects

A structured collection of Java console applications developed to build programming fundamentals. The repository progresses from basic syntax and logic to object-oriented programming, data structures, and external API integration. Many of these projects focus on practical utilities, robotics calculations, and engineering concepts.

## Repository Structure

The projects are organized by complexity into three main categories.

### 1. Simple Projects

Focuses on basic I/O, control flow, loops, and math operations.

- **Math and Logic**: Basic Calculator, Multiplication Table, Odd/Even Checker, Number Guessing Game.
- **Engineering Utilities**: Ohm's Law Calculator, Motor RPM Estimator.
- **Everyday Tools**: Tip Calculator, Age in Days Calculator, Grade Point Averager, Temperature Converter.
- **Strings and Data**: Palindrome Checker, String Reverser, Vowel Counter, Password Strength Checker, Simple Login Prompt.
- **Entertainment**: Dice Roller, Mad Libs Generator.

### 2. Beginner Projects

Introduces arrays, regex, modular methods, and basic state management.

- **Calculators and Converters**: Scientific Calculator, Currency Converter, Unit Converter.
- **Management Systems**: Book Catalog, To-Do List, Vending Machine Simulation, ATM Simulation.
- **Timers and Alerts**: Study Timer, Food Temperature Alert System, Claw Machine Drop Timer.
- **Utilities**: Text Analyzer, Flowchart Syntax Generator, Flashcard Quizzer, Workout Generator.
- **Games**: Rock Paper Scissors.

### 3. Middle-Level Projects

Focuses on Object-Oriented Programming (OOP), file handling, external APIs, and algorithms.

- **Robotics Component Inventory** - Uses File I/O to save, read, and update hardware stock.
- **Local Weather Dashboard** - Integrates with the Open-Meteo REST API to fetch real-time JSON data.
- **Sorting Algorithm Visualizer** - Console-based visualizer comparing Bubble Sort and Selection Sort performance.
- **Tic Tac Toe AI** - Implements the Minimax algorithm for an unbeatable AI opponent.
- **Manufacturing Assembly Line** - Uses Queue data structures to simulate production and quality control.
- **Automated Study Scheduler** - Algorithmically allocates study hours based on available time and priority.
- **Expense Tracker** - Logs data and generates formatted `.txt` budget reports.
- **Claw Machine Arcade Tracker** - Simulates probability and payout-aware grip logic.
- **Food Safety Database Logger** - OOP approach to tracking and updating temperature thresholds.

## Concepts Applied

Across these projects, the following Java concepts are utilized:

- **Core Java**: Variables, loops, conditionals, arrays, strings, regex.
- **Object-Oriented Programming**: Classes, encapsulation, constructors, method overloading.
- **Data Structures**: Lists, Maps, Queues.
- **File I/O**: Reading and writing `.txt` and `.csv` files (`java.nio.file`, `FileWriter`).
- **Networking**: HTTP requests and JSON parsing (`java.net.HttpURLConnection`).
- **Algorithms**: Sorting, Minimax, probability generation.

## How to Run

Ensure you have the [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/downloads/) installed.

1. Clone the repository:

   ```bash
   git clone https://github.com/butunghd0-max/java-learning-projects-type-shyt.git
   ```

2. Navigate to the folder of the project you want to run.

3. Compile the Java file:

   ```bash
   javac Main.java
   ```

   Replace `Main.java` with the specific filename if it differs, e.g., `StudyTimer.java`.

4. Run the compiled class:
   ```bash
   java Main
   ```
